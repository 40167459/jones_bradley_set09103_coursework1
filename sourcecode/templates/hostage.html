Hostage Rescue

Hostage Rescue or Hostage Scenario is a game mode available in all of the Counter-Strike Games. Hostage Rescue Mode is a game mode where the round winning conditions for the two teams are: Counter-Terrorists: Rescue any of the hostages Eliminate all the Terrorists Terrorists Run the round timer down without the Counter Terrorists rescuing any hostages. Eliminate all of the Counter-Terrorists. Maps for this game type are prefixed with cs_.
